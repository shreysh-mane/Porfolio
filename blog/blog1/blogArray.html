Array in javascript

What is an Array?

Suppose you have 5 friends named as "a","b","c","d","e", and you want store their names inside a javascript variable. In basic approach you will do friend1="a"; friend2="b";...friend5="e". to solve this problem javascript introduces a object called Array which enbles the user to store multiple elments inside it.

So Array is a basically a object which allow the user to store the multiple elements inside a sigle variable.

Array Initialization

You can initialize an array with a values in it or you can declare an empty array also.

Note: array can hold elements of different data types

Accessing data form Array

elements inside an array are accessed using index values.

Array index starts form "0" so if you want to get first element of an arry you can get it by array[0]

Find Length of an array

To get the total number of elements inside an array we use length property of an array.
