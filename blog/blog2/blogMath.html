Math Object in javascript

What is an Math obj in js?

We use Math objects to perform operations on numbers

Math.round()

Math.round() rounds the given into the nearest number

Math.pow()

Math.pow() is used of exponential operations

you can either use Math.pow or "**" operator to get the appropriate answer.

Math.sqrt()

Math.sqrt() is used to get square root of the given no

The numbers which are not perfect square will return a floating value

Math.abs()

Math.abs() is used to convert negaitve value into positive value.

Math.ceil()

Math.ceil() is used to convert float value into next integer.

Math.floor()

Math.floor() is used to convert float value into downward first integer.
